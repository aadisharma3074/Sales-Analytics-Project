About This Project

Smart Sales Analytics & AI Profit Prediction System

Project Overview

This project is a Business Intelligence web application developed using Machine Learning and Flask. It helps businesses analyze sales performance, visualize business insights, and predict whether an order will be profitable.

Problem Statement

Businesses generate huge amounts of sales data but often struggle to identify profitable orders, understand customer trends, and make informed business decisions.

Project Objectives

  • Analyze sales performance
  • Visualize business insights
  • Predict profitable orders
  • Generate business recommendations
  • Build an interactive dashboard
